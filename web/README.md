# IT342 Auth System - Web Application

React-based web application for user registration and authentication.

## Features

- User Registration
- User Login
- Protected Dashboard
- User Profile Page
- Logout Functionality

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at http://localhost:3000

## Build for Production

```bash
npm run build
```

## API Integration

The application connects to the backend REST API running on `http://localhost:8080`
